# Tutor Pairing System

A platform that connects students with peer tutors for academic sessions.

## Tech Stack

- Frontend: React.js + Tailwind CSS v4 + React Router v6
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- Real-time: Socket.io
- Payments: Paystack
- File Uploads: Cloudinary
- Email: Nodemailer (Gmail SMTP)
- Deployment: Vercel (Frontend) + Render (Backend)
